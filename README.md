# Takkie Translate Offline

### Privacy-first translation.

A private version of Google Translate. Takkie Translate is a fully offline, locally-run translation tool for macOS. It uses the Google translategemma:12b model to translate text between xxx languages. You can choose not just a language, but also a specific regional variant of a language. No internet or cloud API is required. Your data never leaves your machine. Easy to share with colleagues via AirDrop or by using a thumb-drive - the whole app can be copied to another Mac and just work.
